# Amazon UI/UX Clone

This project is a responsive clone of the Amazon shopping website interface, created for frontend development practice. It focuses on replicating the visual design, layout structure, and basic user experience of the Amazon homepage.

## 🚀 Features

* Responsive design for mobile and desktop 
* Amazon-like navigation bar
* Product grid layout
* Clean and modern UI
* Cross-browser compatible 

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🎯 Purpose

This project was built to improve frontend development skills and understand real-world e-commerce UI structure.

> ⚠️ Disclaimer: This is a personal educational project and is not affiliated with Amazon.
